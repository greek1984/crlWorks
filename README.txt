    Для решения задачи создаётся 2-й массив строк, равный 
по кол-ву элементов исходному.
    Далее в цикле перебираются элементы исходного массива.
Те из элементов, длина которых по условию цикла меньш или 
равна 3-м символам, записываются на позиции 2-го массива.
Они же выводятся на печать в консоли.
